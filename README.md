# Simple Python Slideshow
HTML slideshow that utilizes Python Flask to display images in a customizable interval. 
## Requirements
* Python 3.8+
* [Python Flask](https://pythongeeks.org/python-flask-introduction/ "Python Flask")

## Supported Formats
- .JPG
- .PNG
- .GIF

## Instructions
1. Load script in the same directory as your image folder.
2. Change the variable `READ_DIR` at the top of the script.
3. Run script via CLI
4. Slideshow will stream at `<hostname>:5000`
a. Replace hostname with the IP, or `localhost`if accessing internally

## Tested Hardware
* Raspberry Pi 4 (4GB)
* Raspberry Pi Z
